# What Dinosaur am I?

Web application to determine what kind of dinosaur you are.

## Installation

### Docker Hub

You can easilly pull the latest Docker image and run it with the command `docker pull vojakis/what-dinosaur-am-i`

[Docker Hub repository](https://hub.docker.com/r/vojakis/what-dinosaur-am-i)
